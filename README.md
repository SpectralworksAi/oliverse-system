## How It Works

### MED Console
- Type naturally
- Keystroke rhythm is analyzed in real-time
- ϕ coherence score updates (0-1)
- Flow state (ϕ > 0.85) auto-creates memory

### Memory Vault
- Persistent storage of emotional states
- ϕ-tagged for coherence levels
- Exportable as Pịöŋ JSON

### Export
- One-click JSON export
- Compatible with Ö-Pịöŋ v1.3.0 spec
- Ready for cross-AI transfer
